
# Nodetomic

![alt tag](http://www.developersfeed.com/wp-content/uploads/2016/10/mongodbicon-1.png)

Easy server RESTful, based in MongoDb, Express, Redis, NodeJS.

## Instructions

```bash
git clone [repo Nodetomic]
npm install
```
## Run

command: `npm run serve`

output: http://localhost:8000

## Build

command: `npm run build`

output: `dist`

## Production

folder "dist" ready for production!

```bash
cd dist
nodemon server/app.js
```
output: http://localhost:8001
